# maploops-frontend
maploops-frontend

## Setup Guide

### Prerequisites

- Node 22+
- Angular CLI - 19.0.7+

### Installation

1. **Clone the repository:**
   ```
   git clone https://github.com/octoplusit/maploops-frontend.git
   cd maploops-frontend
   ```

2. **Install npm packages:**
   ```
   npm install
   ```

3. **Start the frontend in dev mode:**
   ```
   npm run start | ng serve
   ```
